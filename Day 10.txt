1. Build a row or column of boxes inside a flex container and control their alignment, spacing, and layout direction using Flexbox.
   
<!DOCTYPE html>
<html>
<head>
  <title>Flexbox Example</title>

  <style>
    .container {
      display: flex;              /* Make flex container */
      flex-direction: row;        /* Change to column if needed */
      justify-content: space-around; /* Horizontal alignment */
      align-items: center;        /* Vertical alignment */
      gap: 15px;                  /* Space between boxes */

      height: 250px;
      border: 2px solid black;
      background-color: #f2f2f2;
    }

    .box {
      width: 80px;
      height: 80px;
      background-color: teal;
      color: white;
      display: flex;
      justify-content: center;
      align-items: center;
      font-size: 18px;
      border-radius: 8px;
    }
  </style>
</head>

<body>

  <div class="container">
    <div class="box">A</div>
    <div class="box">B</div>
    <div class="box">C</div>
  </div>

</body>
</html>

-->display: flex → Activates Flexbox
flex-direction
-->row → horizontal layout
--?column → vertical layout
-->justify-content → Controls horizontal spacing
-->center, space-between, space-around
-->align-items → Controls vertical alignment
-->center, flex-start, flex-end
-->gap → Adds space between boxes

2. Create a web layout with a flex container that holds 3 child boxes.

<!DOCTYPE html>
<html>
<head>
  <title>3 Box Flex Layout</title>

  <style>
    body {
      margin: 0;
      font-family: Arial, sans-serif;
    }

    /* Flex container */
    .container {
      display: flex;
      justify-content: center;   /* Center horizontally */
      align-items: center;       /* Center vertically */
      gap: 20px;                 /* Space between boxes */

      height: 100vh;
      background-color: #e6f2ff;
    }

    /* Child boxes */
    .box {
      width: 120px;
      height: 120px;
      background-color: #4CAF50;
      color: white;
      font-size: 22px;
      display: flex;
      justify-content: center;
      align-items: center;
      border-radius: 12px;
    }

    .box2 {
      background-color: #ff5733;
    }

    .box3 {
      background-color: #3399ff;
    }
  </style>
</head>

<body>

  <div class="container">
    <div class="box">Box 1</div>
    <div class="box box2">Box 2</div>
    <div class="box box3">Box 3</div>
  </div>

</body>
</html>

3. Use the following Flexbox properties to:

        Display the container as a flexbox.

	Set the direction (row or column).

	Adjust horizontal alignment using justify-content.

	Adjust vertical alignment using align-items.

	Add spacing between items using gap.

<!DOCTYPE html>
<html>
<head>
  <title>Flexbox Properties Demo</title>

  <style>
    .container {
      display: flex;              /* 1. Make flexbox */
      flex-direction: row;        /* 2. Set direction (row / column) */
      justify-content: space-evenly; /* 3. Horizontal alignment */
      align-items: center;        /* 4. Vertical alignment */
      gap: 20px;                  /* 5. Space between items */

      height: 300px;
      background-color: #f5f5f5;
      border: 2px solid black;
    }

    .box {
      width: 100px;
      height: 100px;
      background-color: purple;
      color: white;
      display: flex;
      justify-content: center;
      align-items: center;
      font-size: 18px;
      border-radius: 10px;
    }
  </style>
</head>

<body>

  <div class="container">
    <div class="box">1</div>
    <div class="box">2</div>
    <div class="box">3</div>
  </div>

</body>
</html>
