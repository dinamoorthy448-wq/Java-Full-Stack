1.Create a box that takes up 80% of the screen width but no more than 600px.

2.Set a paragraph’s font size to 1.5rem and explain how it behaves on different devices.

3.Use vw to make a heading that scales with screen size.

4.Write a media query that changes background color below 768px.

<!DOCTYPE html>
<html>
<head>
  <title>Responsive Demo</title>
  <style>
    /* 1. Box */
    .box {
      width: 80%;
      max-width: 600px;
      height: 150px;
      background-color: lightblue;
      margin: 20px auto;
      text-align: center;
      line-height: 150px;
    }

    /* 2. Paragraph with rem */
    p {
      font-size: 1.5rem;
    }

    /* 3. Heading with vw */
    h1 {
      font-size: 5vw;
      text-align: center;
    }

    /* 4. Media Query */
    body {
      background-color: white;
    }

    @media (max-width: 768px) {
      body {
        background-color: lightgray;
      }
    }
  </style>
</head>
<body>

  <h1>Responsive Heading</h1>

  <div class="box">Responsive Box</div>

  <p>
    This paragraph uses <b>1.5rem</b> font size.
  </p>

</body>
</html>
