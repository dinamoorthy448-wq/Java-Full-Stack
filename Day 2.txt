1.Understand the difference between block-level and inline elements. Practice examples with <div>, <p> (block) and <span>, <a> (inline).
    Block-Level Elements:
       -->Always start on a new line.
       -->Take full width of the page.
       -->Used to create structure/layout.
    ex:
       <div>, <p>, <h1> – <h6>
    ex code:
       <div>This is a DIV element</div>
       <h1>heading tag</h1>
       <p>This is a paragraph</p>

    Inline Elements
       -->Do NOT start on a new line
       -->Take only the required width
       -->Used inside block elements
     ex:
        <span>, <a>, <strong>, <em>
     ex code:
        <p>This is a <span style="color:red;">red text</span> inside a paragraph.</p>
        <a href="#">Click here</a> to visit website.

2.Learn and practice basic tags: headings <h1>–<h6>, paragraph <p>, division <div>, and inline <span>. Create a sample page using them.
    Headings are used for titles.
       <h1>Main Title</h1>
       <h2>Sub Title</h2>
       <h3>Section Title</h3>
       <h4>Small Heading</h4>
       <h5>Smaller Heading</h5>
       <h6>Smallest Heading</h6>
    Paragraph:
       <p>This is a paragraph of text in HTML.</p>
    Division Used to group content together:
       <div>
         <h2>About Us</h2>
         <p>We provide best services.</p>
       </div>
    Inline <span> Used to style small parts of text:
        <p>I like <span style="color:green;">Web Development</span> and coding.</p>

    Sample HTML Page:
        <!DOCTYPE html>
         <html>
          <head>
           <title>My Practice Page</title>
          </head>

          <body>

           <h1>Welcome to My Website</h1>

          <div>
           <h2>About Me</h2>
           <p>Hello! My name is Dinesh Kumar. I am learning HTML.</p>
          </div>

          <div>
           <h2>My Skills</h2>
           <p>I like <span style="color:green;">Web Development</span> and coding.</p>
          </div>

           <p>Visit my <a href="#">portfolio</a>.</p>

          </body>
          </html>
